
# Evaluación Final Módulo 2: APIs de Películas y SQL.

## Descripción General

Esta evaluación final tiene como objetivo poner en práctica los conocimientos adquiridos en el Módulo 2 del curso de Analista de Datos, enfocados en SQL bases de datos y manejo de APIs. 
Este proyecto se divide en dos fases principales: extracción de datos, modelado de base de datos, análisis mediante SQL y visualización de resultados.

---


El ejercicio consta de dos fases:

## Fase 1: Creación de una base de datos de películas a partir de una API
  
   - Se extraen datos de películas (título, año de lanzamiento, duración, género y contenido adulto) desde un endpoint proporcionado por Adalab.
   - Se crea una base de datos MySQL **desde VS Code**, usando `mysql.connector` en Python para conectar con el servidor MySQL y almacenar la información.
   - Se realizan consultas SQL **desde VS Code** para analizar los datos: duración de las películas, promedio por género, películas por año, contenido adulto, y búsqueda por palabras en títulos o descripciones.

---


## Fase 2: Consultas sobre la base de datos Sakila

   - Se trabaja con las tablas `film`, `actor`, `customer`, `rental` y `category`.
   - Se practican consultas para la realización de los ejercicios.
   - Los ejercicios incluyen conteo de películas por clasificación, promedio de duración, listado de actores en una película, y cantidad de películas alquiladas por cliente o categoría.
